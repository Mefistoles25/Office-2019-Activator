# Office-2019-Activator

Steps to Activate:
1. Go to https://github.com/Nihal247/Office-2019-Activator/blob/main/Activator.cmd, and copy all the code.
2. Create a new text document by click the right button.
3. Open the text document and paste all the code.
4. Go to **File**<**Save As...**.
5. Type the file name as Activator.cmd (you can use any other name, but it should have .cmd in the end!)
6. In Save as type:, select **All Files**.
7. Right click on the file and Run as administrator.
8. Wait 1 minute and let it activate.
9. It will show **Product activation successful**.
10. Close the cmd file and Open any Office app, go to **Account**, you will see **Product Activated**.

Enjoy!
