# Office-2019-Activator

Steps to Activate:
1. Go to https://github.com/Nihal247/Office-2019-Activator.
2. Download the code.
3. Extract the zip file.
4. Right click on Activator.cmd and **Run as administrator**.
5. Wait for a minute and let it activate.
6. Close Command Prompt and open any Office app, say Word.
7. Go to **Account**, and you'll see 'Product Activated'.

Enjoy!
